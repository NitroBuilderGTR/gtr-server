GTR_Server - Patch Notes

Version Release Date: 16th December 2021

GTR_Server v3.01 Open Alpha - Patch Notes

Notes: Major Bug Fixed.

Changes:

Included a Installer with the Required Files needed for Unreal Engine 4 (The Game Engine that GTR_Server uses) to be able to run. There will be a option to install the Required Game Files in Zeon_Launcher. For GTR_Server (no Launcher), just run "Install_Essential_Game_Files.exe", Install the files, and you can now Play GTR_Server.

Bugs

Major Bugs

1) When not running towards Car, you get despawned, and you can't move or use UI.

Solution: Use Run / walk towards the Vehicle when attempting to enter the vehicle. The bug is trying to be fixed.

2) Emergency Respawn Button doesn't work

Solution: If you need to respawn press H. If it doesnt work, please restart your application.

3) FPS / Performance Lag

Solution: Go to The In - Game Settings. You can then turn shadows to the lowest setting. It may look worse, but it gives more FPS. This is recommended for Lower - End PCs / Devices.
